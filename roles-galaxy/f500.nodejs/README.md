Nodejs
======

Install Node.js and NPM

Requirements
------------

Debian Bullseye / Bookworm or Ubuntu Precise/Trusty/Xenial.

Role Variables
--------------

Specify the version of Node.js you want:

    nodejs_version: 6.x

See https://github.com/nodesource/distributions

Example Playbook
-------------------------

    - hosts: servers
      roles:
        - { role: f500.nodejs }
